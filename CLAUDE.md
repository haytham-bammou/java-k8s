# Project Context

## Overview
Java Spring Boot web application (demo project). Fresh project scaffold, just getting started.

## Tech Stack
- **Java**: 21
- **Framework**: Spring Boot 4.0.2
- **Dependencies**: spring-boot-starter-webmvc (Spring Web)
- **Build tool**: Maven (with Maven Wrapper)
- **Packaging**: JAR (default)

## Project Structure
- Group ID: `com.example`
- Artifact ID: `demo`
- Main class: `com.example.demo.DemoApplication`
- Source: `src/main/java/com/example/demo/`
- Tests: `src/test/java/com/example/demo/`

## Build & Run Commands
- Build: `./mvnw clean package`
- Run: `./mvnw spring-boot:run`
- Test: `./mvnw test`
